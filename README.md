# WCAG Audit 

Doe een WCAG audit op een bestaande website uit je eigen omgeving en rapporteer daarover.

De instructies van deze opdracht staan in [INSTRUCTIONS](https://github.com/fdnd-task/wcag-audit/blob/main/docs/INSTRUCTIONS.md).


## The American Book Center

Ik heb de website van The American Book Centeer getest op toegankelijkheid. Dit is de website van een boekenwinkel, waar je ook online boeken kan bestellen.
<img width="1440" height="820" alt="Scherm­afbeelding 2025-11-02 om 16 50 09" src="https://github.com/user-attachments/assets/c15958d0-e6ae-4d4f-b123-39e4a7710d58" />

De website scoorde een 81 op de Lighthouse Test. Dit is best goed. Er zitten vooral schoonheidsfoutjes in de site wat leidt dat het geen 100 is, zoals de benaming van sommige links en buttons. Dit zorgt dat een screenreader het niet voorleest en geen onderscheid kan maken. Voor een volledige WCAG audit [zie wiki](https://github.com/meemo99/wcag-audit/wiki)
